# Iteration 0.0.1 — Contract Manager 首版

## 范围

老板原话：「开发一个合同管理系统，需要登录、用户管理、合同管理，合同附件类型包括pdf、word」

- 登录系统：用户名+密码，session 管理，演示账号
- 用户管理：admin 可增删改禁用用户，重置密码
- 合同管理：CRUD + 状态流转（草稿→待审核→已审批→生效中→已到期/已终止）
- 附件管理：PDF/DOC/DOCX 上传下载，类型校验，大小 ≤10MB
- 操作日志：审计追踪

## 业务模型

见 `requirement_defined` (e000002) payload。

## 产品流程

1. 登录 → 主页（合同列表）
2. 用户管理（admin 专属）
3. 合同 CRUD + 状态流转
4. 附件上传/下载
5. 异常路径：未登录跳转、403/404

## 技术方案

- 后端：Python FastAPI + SQLAlchemy + SQLite
- 前端：Jinja2 模板 + 原生 HTML/CSS/JS
- 认证：Session-based，密码 bcrypt 哈希
- 环境：`~/外部需求/.conda/codingagent` 共享 conda 环境
