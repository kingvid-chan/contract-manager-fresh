# Iteration 0.0.2 — Vue.js 前端 + 暗色主题

## 范围

老板原话（两条）：
1. 「前端修改为vu e」（推断为「前端修改为 Vue」）
2. 「页面样式主题修改为黑色」

- 前端框架从 Jinja2 服务端渲染改造为 Vue.js 单页应用
- 页面整体主题改为暗色（dark mode），黑色背景 + 亮色文字
- 保持 0.0.1 全部功能不变：登录、用户管理、合同 CRUD + 状态流转、附件管理、操作日志

## 业务变更

无。保持 0.0.1 全部业务逻辑和后端 API 不变。

## 技术方案

- 前端：Vue.js 3 + 原生 HTML/CSS/JS（不引入构建工具链，CDN 加载 Vue）
- 主题：暗色配色方案，CSS 变量驱动
- 后端：FastAPI API 保持不变，静态文件服务改为 SPA fallback
- 路由：Vue Router hash mode
- 通信：Axios 或 fetch，保持现有 Session 认证

## 产品流程

同 0.0.1，交互从页面跳转改为 SPA 路由切换。

## 数据边界

同 0.0.1：仅演示账号和假数据。

## 自测要求

- 覆盖全部 0.0.1 功能在新前端下的回归测试
- 暗色主题各页面的视觉快照
