# Iteration 0.0.2 — Vue.js 前端 + 暗色主题

## 范围

老板原话（两条）：
1. 「前端修改为vu e」（推断为「前端修改为 Vue」）
2. 「页面样式主题修改为黑色」

- 前端框架从 Jinja2 服务端渲染改造为 Vue.js 单页应用
- 页面整体主题改为暗色（dark mode），黑色背景 + 亮色文字
- 保持 0.0.1 全部功能不变：登录、用户管理、合同 CRUD + 状态流转、附件管理、操作日志

## 技术方案

- 前端：Vue.js 3 + 原生 HTML/CSS/JS（不引入构建工具链，CDN 加载 Vue）
